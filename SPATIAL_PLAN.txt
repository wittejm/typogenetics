│ Spatial Typogenetics Soup — Implementation Plan                                                                                │
│                                                                                                                                │
│ Context                                                                                                                        │
│                                                                                                                                │
│ The typogenetics project has a working "mixed" (well-mixed) stochastic soup where any strand can react with any other strand.  │
│ Findings show the soup collapses to a C/G monoculture due to lack of structural constraints. Adding spatial locality — strands │
│  on a 2D grid that can only react with neighbors — is the most promising extension for producing qualitatively different       │
│ emergent behavior (local ecosystems, boundaries, cluster persistence).                                                         │
│                                                                                                                                │

│ Step 2: Grid data structure (src/spatial/grid.ts)                                                                              │
│                                                                                                                                │
│ Flat array of cells: string[][] where grid[cellIndex] is the strands in that cell. Index arithmetic: idx(row, col, width) =    │
│ row * width + col.                                                                                                             │
│                                                                                                                                │
│ Functions:                                                                                                                     │
│ - idx, rowCol, neighbors — coordinate helpers                                                                                  │
│ - initGrid(strands, width, height) — distribute initial strands randomly across cells                                          │
│ - diffuse(grid, width, height, rate) — each strand has probability rate of moving to a random adjacent cell. Mutates in place. │
│ - enforceCrowding(grid, width, height, cellCapacity) — cells over capacity push excess to random neighbor; if neighbor full,   │
│ strand is evicted. Mutates in place.                                                                                           │
│ - computeDensity(grid) — returns number[] of strand counts per cell                                                            │
│ - flattenGrid(grid) — returns all strands as a single string[] for global stats                                                │
│                                                                                                                                │

│ Step 4: Spatial worker (src/spatial/worker.ts)                                                                                 │
│                                                                                                                                │
│ Same structure as src/soup/worker.ts. Key difference in the reaction loop:                                                     │
│                                                                                                                                │
│ 1. Pick a random non-empty cell                                                                                                │
│ 2. Pick a random source strand from that cell                                                                                  │
│ 3. Translate → get enzymes                                                                                                     │
│ 4. Pick random enzyme                                                                                                          │
│ 5. Collect all strands from same cell + adjacent cells as candidate targets (with cell/index tracking so we can remove from    │
│ the correct cell)                                                                                                              │
│ 6. Pick target from this local pool, find binding positions                                                                    │
│ 7. Execute reaction (reusing model/ functions)                                                                                 │
│ 8. Remove target from its cell, add products to source's cell                                                                  │
│ 9. After each batch: run diffuse() and enforceCrowding()                                                                       │
│                                                                                                                                │
│ Messages: same protocol as soup worker (start/stop/pause/resume/config), with start carrying gridWidth/gridHeight and stats    │
│ including density array.                                                                                                       │
│                                                                                                                                │
│ Step 5: Grid canvas (src/spatial/GridCanvas.tsx)                                                                               │
│                                                                                                                                │
│ Canvas element that draws a heatmap from the density array. Color ramp: white (empty) → yellow → orange → red (at capacity).   │
│ Repaints in useEffect when density changes, throttled via requestAnimationFrame. Canvas dimensions: 480x480 with               │
│ image-rendering: pixelated. Mouse hover shows cell coordinates + strand count.                                                 │
│                                                                                                                                │
